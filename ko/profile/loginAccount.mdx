직원 계정을 생성하고 기존 계정을 수정할 수 있습니다.

해당 버튼이 없다면 로그인한 계정의 권한이 없기 때문입니다.

1. 하단 **\[프로필 관리\]** 선택
2. 우측 상단 **\[직원 관리\]** 클릭

---

## 계정 생성

해당 버튼이 없다면 로그인한 계정의 권한이 없기 때문입니다.

<img
  src="/images/프로필관리_직원관리.png"
  alt="Image"
  title="Image"
  style={{ width:"59%" }}
/>

1. **\[직원 계정 추가\]** 버튼 클릭
2. 아래 정보를 입력합니다.

![Image](/images/프로필관리_직원계정생성.png)

1. 계정 ID
2. 계정 패스워드
3. 이름
4. 계정 닉네임
5. 연락처
6. 이메일

## 계정 수정

직원 계정의 비밀번호를 모를 때 변경할 수 있습니다.

1. 직원 목록에서 비밀번호를 변경할 계정을 클릭
2. \[계정 패스워드\] 란에 새로운 비밀번호를 기재
3. \[패스워드 확인\] 란에 동일한 비밀번호를 기재
4. 사용할 직원에게 공유.

---

## 계정 삭제

1. 직원 관리 목록에서 해당 계정 확인
2. 우측 **\[권한 삭제\]** 클릭
3. 삭제 확인 팝업에서 **\[삭제\]** 선택

※ 삭제된 계정은 복구할 수 없습니다.
