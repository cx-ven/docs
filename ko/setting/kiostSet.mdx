## 시스템 설정

설정 항목

- 자동 청소지시 비활성화
- 차량 출차 버튼 비활성화
- 차량 방문 여부 확인
- 현장 판매 활성화

## 미성년자 차단 기능

1. 선택 인원 전원 성인 인증 : \
   • 활성화 : AI 성인 예측과 무관하게 투숙자가 키오스크에서 선택한 투숙 인원 수 대로 신분증 검사를 요구합니다.\
   • 비활성화 : AI 성인 예측 후 성인이 아님으로 판단될 때 1명의 신분증 만 요구합니다.
2. 약관 동의 추가 안내문 : \
   인원 수 선택 후 신분증 검사 전 업장 별 추가 동의 안내문을 띄울 수 있습니다.
3. CCU 타입 : 해당 업장이 CCU 사용 중인지 아닌지 확인할 수 있습니다.
4. 키오스크 배너 업로드 : 키오스크 하단 배너를 직접 등록할 수 있습니다.
5. 키오스크 예약체크인 옵션 순서 선택 : 키오스크 예약 체크인 시 예약을 조회할 수 잇는 순서를 직접 설정 할 수 있습니다.\
   QR체크인 - 벤디트 클라우드에 예약 생성 시 자동문자 기능을 사용하는 업장에 한하여 투숙 고객님께 체크인 QR문자가 발송됩니다.
   해당 QR로 키오스크를 통해 보다 편리한 체크인이 가능합니다.
6. 숙박 현장 판매 기준 : \
   • 예약 재고 기준 (예약 → 당일 숙박 재고를 바라봅니다.)\
   • 공실 기준 (대시보드 → 당일 공실 객실을 바라봅니다.)
7. 예약자명 유사도 검색 허용 :
   - 유사도 검색 허용 OFF : 예약자명을 100%로 정확하게 입력해야 키오스크에서 예약 조회 가능
     - 띄어쓰기, 구분자, 성조 제거만 지원
     - 성/이름 이름/성으로 조회 지원하지 않음
   - 유사도 검색 허용 ON : 유사도율 조정이 가능
     - 하높음(0.8) / 중간(0.6) / 낮음 (0.4)
   - 예약자 명이 일본어,중국어,영어인 경우 → 영어로 입력
   - 예약자 명이 한국어인 경우 → 유사도율과 별개로 100%로 일치하도록 입력하여야 조회 가능,영어로 입력 경우는 스펠링 입력 후 발음으로 추가 검증이 이루어짐
8. 예약 2차 인증:\
   • 유사도와 관련 없이 예약 조회 시 예약 2차 인증 여부 설정 가능\
   • 키오스크를 통해 예약자 명으로 조회 및 예약 선택 시 본인의 예약 건이 맞는지 확인여부를 위해 고객이 예약자 명을 2차적으로 확인할 수 있는 화면이 우측 사진과 같이 노출됩니다
9. 영수증 발급 조건: 키오스크를 통해 결제 시 영수증 출력 설정을 \[고객이 출력버튼을 눌러 출력\] 또는 \[결제 및 예약시 항상 출력\] 중 선택 할 수 있습니다.
10. 체크인시 예약 정보 출력 :예약 정보(예약자명, 벤디트 예약번호, 연락처, 객실유형, 예약경로)가 영수증에 함께 출력됩니다.
11. 체크인시 객설 번호 미출력 :영수증 출력 시 입실 정보 (사업장 정보, 입실시간, 퇴실시간)만 출력되며 객실 번호는 출력되지 않습니다.
12. 방출기 지폐 권종 :\
    • **1번 현금 방출기** - 키오스크를 정면으로 봤을 때 왼쪽 함의 권종 설정\
    • **2번 현금 방출기** - 키오스크를 정명으로 봤을 때 오른쪽 함의 권종 설정설정 가능한 권종은 5만원, 1만원, 5천원, 천원 입니다.두 방출기의 권종이 동일한 경우 오류가 발생 될 수 있습니다. 다른 권종으로 설정 부탁드립니다.
