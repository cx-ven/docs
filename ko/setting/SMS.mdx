# 자동 문자/예약문자 사용하기

예약 정보를 기반으로 사용자가 설정한 메시지 양식을 원하는 투숙 시점에 자동으로 문자를 발송하는 서비스입니다.

대시보드 우측 하단 [설정] → 우측 메뉴 [자동 문자]를 선택한 후 원하는 발송 양식, 발송 단계 등을 상세히 설정할 수 있습니다.

[사용 중인 양식] 하위에 현재 자동 문자 발송에 사용되고 있는 양식을 한 눈에 확인할 수 있습니다.

용할 메시지 양식을 빠르게 활성/비활성화하거나 ‘양식 추가하기’를 누르면 [메시지 양식 설정] 화면으로 바로 이동할 수 있습니다.

메시지 양식을 사용으로 설정한 이후에 새로 생성되는 예약부터 문자 발송이 적용됩니다.

이미 생성되어 있는 예약에는 자동 문자가 발송되지 않습니다.

#### 알림톡 양식 설정 (기본양식) 설정

- ‘기본 양식’은 메시지 내용이 고정된 ‘카카오 알림톡’ 양식을 말합니다.
- 카카오 알림톡은 정책상 심사를 통과한 메시지 내용으로만 발송할 수 있어 내용 편집이 불가합니다.
- 카카오 알림톡은 발송 시각, 발송 시점만 변경하여 사용할 수 있습니다.
- 알림톡 메시지 내용 하단 [문자양식으로 복사하기]로 간편하게 동일한 내용의 문자 메시지 양식을 추가할 수 있습니다. 문자 메시지 양식에서는 내용을 자유롭게 편집할 수 있습니다.
- 하단 메시지 양식 설정 하단에 [카카오 알림톡 발송 실패 시] 옵션을 통해 ‘문자로 대체 발송’ 또는 ‘발송 실패(잔액 미차감)’ 처리할 수 있습니다.
- 카카오 정책으로 인하여 안심번호는 카카오 알림톡 발송이 불가합니다.

#### 문자 메시지(자유양식) 설정

- 문자 자유 양식은 원하는 내용으로 자유롭게 편집하여 문자 메시지로 발송할 수 있습니다.
- 기존에 ‘사용 중’인 메시지 내용을 편집하고 ‘저장’을 누르면, ‘저장’을 완료한 시점부터 변경된 내용으로 자동 발송됩니다.
- 변경하신 설정 적용 시점은 동시간 발송량에 따라 최대 3분까지 소요될 수 있습니다. 변경된 내용은 발송 내역을 참고해 주시기를 바랍니다.
- 양식 설정 시 우측 미리보기를 통해 발송될 내용, 메시지 종류, 글자 수를 미리 확인할 수 있습니다.

#### 자동입력

`@업장명`, `@예약번호` 등 자동입력을 클릭하여 메시지 내용에 추가합니다. 자동 입력 데이터는 실제 발송 시에는 예약 정보로 자동 변경되어 발송됩니다.

#### 누구에게 발송이 되나요?

클라우드 예약 상세정보에 기재된 연락처로 자동문자가 발송됩니다.

수신 가능한 국내 연락처 또는 안심번호 연락처가 있다면 설정한 자동 문자 발송이 됩니다.

자동 문자는 국내 번호만 발송이 가능합니다.

업장이 설정한 메시지 양식을 해외 연락처, 이메일 발송은 불가합니다.

안심번호는 문자 양식만 발송이 가능합니다.

### 자동문자 발송 요금

현재 지원하는 메시지 종류 및 발송 요금은 아래와 같습니다.

| 카카오 알림톡    | 띄어쓰기 포함, 한글 1,000자 | 띄어쓰기 포함, 영어 1,000자 | 건당 30원 | 벤디트           |

- 자체 알림톡, 자사 비지니스 카카오톡 채널로 발송되는 기능은 지원되지 않습니다.

## 잔액 충전 / 자동 충전 관리

자동 문자 발송에 필요한 잔액 관리를 할 수 있습니다.

잔액 충전은 결제 카드를 등록하여 일정 잔액 미만일 때 자동으로 충전하거나, 다른 결제 수단 (신용/체크카드, 간편결제(네이버페이/카카오페이/페이코), 가상계좌)으로 가능합니다.

\*설정한 [자동충전 잔액기준] 보다 현재의 잔액이 낮으면 자동 충전되지 않기 때문에 수동으로 충전하여 사용해주세요.

#### 자동충전 카드 등록 실패하는 경우

계정의 권한 문제로 카드 등록이 안될 수 있습니다. 점주님의 아이디로 클라우드에 접속하여 카드를 등록해주세요.

지속 실패하는 경우 실패 화면을 캡쳐하여 기술지원으로 문의해주세요.

#### -10,000원까지 사용 가능한 이유

발송 대기 중인 문자 등, 여러 상황을 고려하여 미수금액 -10,000원까지 자동 문자 서비스를 이용할 수 있습니다.

정상적인 서비스 사용을 위해 -10,000원을 포함한 금액을 수동으로 충전해주세요

#### 잔액 소진 알림톡 수신

클라우드 내 [설정] → [계정 및 일반] → [알림문자 수신번호]에 추가한 연락처로 서비스 이용 알림톡이 발송됩니다.

이용 알림톡 (최소 잔액 도달 시, 자동 충전 완료 시, 자동 충전 실패 시, 미납금(마이너스 금액)발생 시, 잔액 소진 시)

- 서비스 이용 알림톡은 잔액 차감 대상이 아닙니다.

## 발송 내역

발송 건별로 발송일과 발송 결과, 메시지 내용 등을 확인할 수 있습니다.

발송 내역은 최대 1년 치 자료가 보관됩니다. (최대 1개월 단위로 조회 가능)

- **수신 번호, 예약 번호에 대한 발송 내역 검색이 가능합니다.**

번호 입력 시 연속된 숫자 4자 이상을 입력해야 검색이 가능합니다.

하이픈이 포함되어도 하이픈은 삭제되고 숫자만 입력/검색됩니다.

## 환불(결제 취소) 가능 기한

충전 내역 에서 환불(결제 취소) 가능 기한에 해당하는 결제 건에 대해 건 당 결제 취소가 가능합니다.

- 결제일을 포함하여 180일 이내에만 결제 금액의 전액/부분 취소가 가능합니다.
- 결제일을 포함하여 180일이 지난 경우, 결제 취소가 불가합니다.

#### 결제 금액 부분 취소

환불(결제 취소)는 전액 취소, 부분 취소가 가능합니다.

\*가상 계좌는 전액 취소만 가능합니다.

전체/부분 취소 모두 최초 결제 건의 결제일 포함 180일 이내에만 가능합니다.

#### 결제 수단 별 취소 기한 및 소요 기간

| 결제 수단                   | 취소 기한          | 취소 소요 기간                                                                                          |
| :---------------------- | :------------- | :------------------------------------------------------------------------------------------------ |
| 신용 카드                   | 결제일 포함 180일 이내 | 매입 전 취소(승인 취소/전체 취소) : 취소 요청 직후 즉시  매입 전 취소(부분 취소) : 영업일 기준 3~~4일  매입 후 취소 : 취소 요청 후 영업일 기준 3~~4일 |
| 간편결제(네이버페이, 카카오페이, 페이코) | 결제일 포함 180일 이내 | 결제 수단으로 등록한 카드, 계좌의 소요 기간과 동일                                                                     |
| 가상 계좌                   | 결제일 포함 180일 이내 | 영업일 기준 2~3일 소요                                                                                    |
| 체크 카드                   |                | 최대 10일 정도가 소요  전액 취소의 경우 취소 요청 당일 환불                                                              |

### 서비스의 이용 중지(해지)

클라우드/VPMS의 이용 중지(해지)를 원하실 경우에는 고객센터로 별도 요청해 주셔야 합니다.

서비스의 이용 중지(해지) 수수료는 현재 잔액의 5%입니다. 수수료 5%를 제외한 나머지 금액에 대하여 결제건 환불(결제 취소) 처리가 가능합니다.

\*‘문자 발송’ 서비스만 별도로 이용 중지(해지)는 불가합니다. 문자 발송을 일시 중단하실 때는 ‘사용 중인 양식’을 모두 ‘미사용’ 처리하시면 잔액이 차감되지 않습니다.

  [예약 생성 시]는 요금의 설정 시간 기준이 아닌, 해당 예약이 실제로 생길 때를 일컫습니다.

<Accordion title="Q. 수동 예약은 연락처 입력해야만 발송되나요?">
  업장에서 수기 예약 생성 시 연락처가 있어야하며, **연락처로 자동문자 발송** 체크 박스에 체크를 해주셔야합니다. 옵션 체크 시에만 사용 중인 자동 문자 양식이 발송됩니다
</Accordion>

<Accordion title="Q. 발송 조건을 선택 할 수 있나요?">
  특정 플랫폼, 특정 사용자 등 발송 조건을 필터하여 자동 문자를 발송 할 수는 없습니다.
</Accordion>

<Accordion title="Q. 자동입력 변수로 양식 설정하는 방법">
  문자를 받는 사람마다 자동으로 내용이 변경되어야하는 내용이 있다면 “자동 입력” 하위에 있는 @으로 넣어주셔야 합니다.
</Accordion>

<Accordion title="Q. 자동문자를 재발송하고 싶어요.">
  클라우드 하단 [예약] → 재발송을 원하는 예약 건 선택 → 연락처 옆 [예약문자 재발송]버튼으로 재발송 할 수 있습니다.
</Accordion>

<Accordion title="Q. 특정 날짜·시간(입실 당일 오후 1시)에만 보내고 싶습니다">
  문자 양식에서 우측 [발송 단계] → [발송 시각]에서 발송 단계를 특정 시각으로 설정할 수 있습니다.
</Accordion>

<Accordion title="Q. 문자 발송 대기 중인데 취소할 수 없나요?">
  예약 문자로 발송되기 때문에 발송 대기 중인 문자는 예약이 취소되지 않다면 취소가 어렵습니다.
</Accordion>

<Accordion title="Q. 문자가 안 온 경우 업장에서 어떻게 처리해야 하나요">
  1. 예약에 전화번호가 기입되어있는지 확인해주세요.
  2. 사용자가 문자를 수신 할 수 있는 장소가 아니거나, 차단한 문자에 있는지 확인해주세요. 그럼에도 문자 수신을 하지 못하셨다면, 예약자 명과 번호를 기술지원 부서에 전달하여 문의해주세요.
</Accordion>

<Accordion title="Q. 벤디트의 '예약 번호'로 체크인을 위해선 문자 발송이 불가피한데, 비용이 발생하지 않는 체크인 방법이 있나요?">
  '예약자명으로 체크인'하시면 문자 안내/발송 없이 바로 예약 체크인이 가능합니다.

  문자 발송을 통해 예약 체크인이 가능한 방식은 '예약 번호' 와 'QR코드' 입니다.
</Accordion>
