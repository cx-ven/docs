## 객실 자동 배정 방식

<img
  src="/images/설정_객실자동배정방식.png"
  alt="Image"
  title="Image"
  style={{ width:"95%" }}
/>

- 타입별 우선순위에 따름 : 객실 타입 별 우선 순위에 따라 예약 고객 객실이 배정되는 방식입니다.\
  설정 → 객실 도움말에 표시되어 있는 각 타입 하위에 표시된 객실의 표시 순서와 같습니다.\
  같은 타입 좌측 상단부터 시작하여 먼저 표시되는 객실이 우선 배정되도록 설정합니다.
- 최장공실객실 우선 : 객실이 공실으로 방치된지 오래된 순서로 먼저 배정하는 방식입니다.
- 공실객실 중 랜덤배정 : 공실 객실 내에서 랜덤으로 배정되는 방식입니다.

## OTA 연동 관련 설정

<img
  src="/images/설정_OTA연동설정.png"
  alt="Image"
  title="Image"
  style={{ width:"54%" }}
/>

- 다중지점 이용 시 지점 정보가 모호한 연동 내역 무시여러 지점은 운영하시는 경우 해당 체크 설정 시 지점 정보가 모호한 연동은 연동되지 않도록 설정 할 수 있습니다.
- OTA 연동 예약생성 기본값예약 문자에 시간 정보가 전달되지 않는 종류의 OTA에서 전달되는 연동 내역에 대해 기본 체크인 / 아웃 시간을 지정할 수 있습니다.

    1. 자동 배정 기능이 체크되어있다면 [N분] 이내에 입실할 미리 예약 중 아직 객실이 배정되지 않은 예약에 대해 시스템이 자동으로 객실을 배정합니다.이때 자동으로 배정되는 객실의 우선순위는 예약고객 객실배정 설정을 따릅니다.
    2. 예약 입실시간 [N분] 전에 체크인 시도 시 체크인이 가능합니다.
  </Accordion>
  <Accordion icon="3" title="이용객 체크아웃 시간 [N분] 전 객실 카드키 제거시 퇴실로 판단">
    CCU 사용 시 [N분]내에 객실 카드키 제거가 되어 키오스크를 통해 체크아웃 시 정상적인 체크아웃이되며, N분 이전에 체크아웃 시 사용 중으로 판단하여 객실 키만 회수합니다.
  </Accordion>
  <Accordion icon="4" title="(RCU미연동 시) 객실 판매 시 기본 체크인으로설정">
    체크시 키오스크에서 방 판매시 및 특정 객실에 숙박/대실정보 생성시 객실 상태의 기본값을 사용중으로 표시합니다.객실의 체크인 여부를 확인할 수 없는 RCU 미연동 매장에서 사용 가능한 기능입니다.
  </Accordion>
  <Accordion icon="5" title="이상 사용기록 감지기능 활성화">
    체크시 사용중(미상) 상태의 사용기록이 30분 이상 지속될 경우 사용이 끝난 뒤 이상 사용기록을 예약으로 남겨 저장합니다.
  </Accordion>
  <Accordion icon="6" title="숙박 체크인 시간 제한 없음">
    체크시 방문 손님이 숙박 시작일에는 어느 시간에나 키오스크를 통해 예약 입실이 가능합니다. (얼리체크인 제한이 없어집니다.)
  </Accordion>
  <Accordion icon="7" title="대실 체크인 시간 제한 없음">
    체크시 방문 손님이 체크인 시간보다 늦거나 일찍 체크인하는 경우 입실 시간에 맞춰 퇴실 시간이 자동 재조정됩니다.
  </Accordion>
  <Accordion icon="8" title="예약 수동배정시 입실/퇴실시간 자동조정">
    해당 기능 활성화 시  대시보드에서 직접 다가오는 예약을 객실에 배정할 때, 배정하는 시점을 기준으로 숙박으로 변경되고 입실시간에 맞춰 입/퇴실시간을 재조정합니다.
  </Accordion>
  <Accordion icon="9" title="투숙중인 객실 재배정시 자동 청소지시">

  </Accordion>

  </Accordion>
