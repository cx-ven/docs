# PMS 계정 로그인

- [내 지점]의 업장 이름 클릭🖱️
- 여러 개의 업장을 관리하는 계정인 경우, 목록에서 원하는 지점을 선택하면 해당 지점의 대시보드로 이동합니다.

<AccordionGroup>
  <Accordion title="로그인 계정, 로그인 비밀번호를 잊어버렸을 경우">
    업장 내 로그인 계정을 아는 사람이 없는 경우 💬기술지원(채팅상담링크 넣기)으로 문의해주세요
  </Accordion>
  <Accordion title="클라우드(PMS) 로그인 계정 비밀번호 변경하는 방법">
    1. 하단 [프로필관리]🖱️클릭 (프로필관리 메뉴가 없는 경우 권한이 없기 때문입니다.)
    2. 우측 메뉴 [직원관리]🖱️클릭
    3. 변경이 필요한 [아이디]🖱️클릭
    4. 계정 패스워드에 새로운 비밀번호 입력,  패스워드 확인에 동일하게 입력
  </Accordion>
</AccordionGroup>

# 모바일로 클라우드(PMS) 접속 방법

갤럭시(안드로이드,And)는 플레이스토어에서 앱을 다운 받아 사용 가능합니다.

아이폰(IOS)는 별도 앱 지원을 하지 않습니다. \
사파리에서 URL(https://cloud.vendit.co.kr/)로 접속하여 사용합니다.

<Accordion title="V-Cloud 앱 다운 받는 방법 (안드로이드만 지원, ios 앱 미지원)">
  1. 구글 플레이스토어 접속
  2. 🔎벤디트 검색
  3. "벤디트 V-Cloud"설치
</Accordion>

---

### 간편 로그인이란?

벤디트 V-Cloud 앱(안드로이드만 지원) 로그인 시 사용하는 기능입니다.

<Accordion title="간편 로그인 기능 사용 방법">
  1. 앱에서 간편 로그인 버튼 클릭 → 일회용 로그인 번호가 생성
  2. PC 클라우드 → 설정 → 계정 및 일반 → 핀 번호로 로그인
  3. 발급된 핀 번호 입력
  4. 모바일 기기 자동 로그인 완료
</Accordion>
