# FAQ

<Accordion title='"얼리 체크아웃" 이 뭐에요?'>
  퇴실 예정일 20시간 전 키오스에서 키 반납 시 “얼리 체크아웃”으로 퇴실처리를 합니다.
</Accordion>

<Accordion title='"000호 체크아웃 없이 객실키만 회수함"이 있고 체크아웃이 안됐어요'>
  키오스크에서 객실 키가 **체크아웃으로 회수되더라도**, 아래 조건에 해당하는 경우 **객실 상태는 변경되지 않고 키만 회수됩니다.**

  - 객실이 **재실 상태**인 경우
  - 객실이 **공실 상태**인 경우
  - 객실 내 **키 슬롯에 키가 꽂혀 있어 재실로 판단되는 경우**
</Accordion>

<Accordion title='"결제 후 고객 이탈(시간 초과)"이 뭐에요?'>
  현장 방문객이 결제 후 체크인을 위한 [다음] 버튼을 누르지 않고 홈 화면으로 돌아가거나, 또는 아무런 액션을 취하지 않을 경우 정상적인 예약이 만들어지지 못했기 때문에 해당 기록이 남게됩니다.\
  자동 환불은 불가하여 필요 시 별도 환불처리가 필요합니다.
</Accordion>

<Accordion title="방출기 카드 없음">
  키오스크에 방출할 수 있는 키가 없을 때 알림이며 키오스크 함체를 열어 추가 객실 키를 적재해주세요.
</Accordion>

<Accordion title="방출기 회수통 가득참">
  키오스크 회수함에 키가 가득 찬 경우 키오스크에서 키 발행/키 회수가 불가합니다.\
  키오스크 하부장을 열어 적재함 밑에 있는 회수함에서 키를 제거해주세요.
