운영 객실 수/객실 상태/객실 타입 별로 원하는 객실을 필터하여 조회 할 수 있습니다.

> - [ ↻ ] 아이콘 클릭 시 조회 조건을 초기화 합니다.
> - [펼치기 ^] / [접기 ^] 클릭하여 객실 조회 조건을 노출 / 숨김 처리합니다.

---

# 공통 객실 상태

1. 객실들 및 해당 객실들에 배정된 예약의 상태가 **객실 카드**로 표시됩니다.

<Card title="상태별 탭 가이드">
  <img
    src="/images/대시보드_공통객실상태.png"
    alt="Image"
    title="Image"
    className="mr-auto"
  />

  <Tabs>
    <Tab title="공실">
      현재 예약이 배정되어 있지 않고, 즉시 사용 가능한 객실입니다.  공실 객실 중 요금 설정을 통해 현재 시간에 판매 요금이 책정되어 있는 공실 객실이 키오스크 판매 화면에 노출됩니다.
    </Tab>
    <Tab title="청소">
      청소를 지시한 객실은 청소지시 상태로 표시됩니다. 실제 청소부가 입실하여 청소를 진행 중인 객실은 청소중 상태로 강조 표시됩니다.
    </Tab>
    <Tab title="숙박 ">
      숙박 예약이 입실하여 이용중이거나 입실 대기 상태인 객실은 숙박중 혹은 입실대기 상태로 표시됩니다.
    </Tab>
    <Tab title="대실">
      대실 예약이 입실하여 이용중이거나 입실 대기 상태인 객실은 대실중 혹은 입실대기 상태로 표시됩니다
    </Tab>
    <Tab title="선택/결제 중">
      키오스크에서 해당 객실을 선택 후 결제 단계에 있을 때 표기 됩니다.

      - 비정상적인 방법으로 결제 도중 벗어나는 경우, 자동 공실 변경이 되지 않아 클라우드에서 수동으로 공실 변경이 필요합니다.
    </Tab>
    <Tab title="이용만료">
      예약 이용 시간이 지난 객실은 [이용만료] 상태로 표시됩니다. \
      설정 -\> 예약 설정에서 설정한 객실 카드키 제거시 퇴실로 판단 설정에 따라이용 만료 시각이 임박했고, 객실 카드키가 제거된 상태의 객실의 경우 [퇴실확인] 상태로 표시됩니다.
    </Tab>
    <Tab title="판매중지">
      객실을 선택 후 판매중지 상태를 선택했거나, 설정 -\> 판매 및 마일리지 설정에서 판매 중지 상태를 선택한 객실들은 회색 판매중지 상태로 표시됩니다.  \
      현재 입실 중인 객실, 청소중인 객실도 판매 중지할 수 있습니다.
    </Tab>
    <Tab title="숙박(장기)">
      수기 예약 생성 시 장기 숙박으로 선택할 수 있습니다.\
      숙박과 장기 숙박을 판별하기 위해 노출되는 색상이 다릅니다.
    </Tab>
  </Tabs>
</Card>

2. 추가로 클라우드와 CCU 연동 여부에 따라 상태 확인 방법이 다릅니다. 아래 순서에 따라 확인해 주세요.

# 내 업장이 CCU를 사용하는지 확인하기

클라우드 → 설정 → 키오스크 설정 → 좌측에서 사용하는 첫 번째 키오스크 선택 → [CCU 타입]

<Callout>
<Accordion defaultOpen icon="link-slash" title="“CCU 미사용”">
    ![Image](/images/대시보드_CCU미사용객실상태.png)

    <Tabs>
      <Tab title="이용만료">
        객실 이용 중  또는 외출중 상태에서 현재 시간 범위의 매칭되는 예약이 없는 상태- 체크인 처리 후 배정취소 or 배정 객실을 옮겼을 경우 \
        → 객실 확인이 필요한 객실로 간주하고 바로 공실이 아닌 “이용 만료”로 상태를 변경하게 됩니다.

        - 퇴실 시간이 지났으나 키오스크에서 체크아웃 처리 하지 않았을 경우
        - 예약의 입/퇴실 일자를 잘못 수정해서 현재의 예약이 아닌 상태로 배정됐을 경우
      </Tab>
      <Tab title="청소 지시">
        키오스크를 통해 체크아웃 시 → 자동으로 청소지시로 변경 됨
      </Tab>
      <Tab title="점검 지시/점검 중">
        클라우드 → 설정 → 키오스크 설정 → 객실 점검 비활성화 “해제”

        - 대시보드 → 객실 클릭 → 점검 지시 수동 선택
        - 키보드 단축키로 상태 변경 : 점검지시 : shift + Q / 점검중: shift + W
      </Tab>
      <Tab title="외출중">
        수기로 상태 변경 필요
      </Tab>
    </Tabs>
</Accordion>
</Callout>
