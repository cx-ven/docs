# 다가오는 예약이란?

오늘 입실 예정이며 객실이 미배정된 예약이 있는 경우, **대시보드 \> 다가오는 예약** 영역에서 예약 카드를 확인하고 객실을 배정할 수 있습니다.

오늘 체크인 예약과 현재 시점 기준 24시간 이내 체크인 예정 예약만 노출됩니다.

---

# 예약 배정부터 룸 체인지 방법 배우기

아래 기능을 배울 수 있습니다.

1. 다가오는 예약 배정
2. 강제 배정 (예약 객실과 배정 객실의 타입이 다를 때)
3. 체크인 전 룸 체인지

예약 카드를 이동할 때 객실 상태에 따라 다음과 같이 표시됩니다.

<Callout icon="bird">
  - 매우 어두움: 배정 불가 객실 (놓아도 적용되지 않음)

  - 약간 어두움: 배정 가능하나 객실 타입 불일치 → 배정 시 확인 창이 표시됨

  - 밝게 표시: 타입 일치 및 배정 가능

  - 공실: 즉시 배정

  - 기존 예약 있음: 예약 병합 화면 표시 후 확인 시 병합 처리
</Callout>
