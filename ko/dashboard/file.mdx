# 다가오는 예약 확인

오늘 입실 예정이며 객실이 미배정된 예약이 있는 경우, **대시보드 \> 다가오는 예약** 영역에서 예약 카드를 확인하고 객실을 배정할 수 있습니다.

오늘 체크인 예약과 현재 시점 기준 24시간 이내 체크인 예정 예약만 노출됩니다.

---

# 다가오는 예약 배정하기

![Image](/images/대시보드_다가오는예약배정하기.png)

- 배정할 **예약 카드를 마우스 왼쪽 버튼으로 클릭 후 드래그**합니다.
- 원하는 **객실 카드 위에 놓으면 배정**됩니다.

  ### 객실 상태별 배정 시 표시 기준

  예약 카드를 이동할 때 객실 상태에 따라 다음과 같이 표시됩니다.

  > **- 매우 어두움**: 배정 불가 객실 (놓아도 적용되지 않음)
  >
  > **- 약간 어두움**: 배정 가능하나 객실 타입 불일치\
  > → 배정 시 **확인 창**이 표시됨
  >
  > **- 밝게 표시**: 타입 일치 및 배정 가능
  > - 공실: 즉시 배정
  > - 기존 예약 있음: **예약 병합 화면** 표시 후 확인 시 병합 처리

## 예약 재배정 (체크인 전)

체크인 전 이미 객실이 배정된 예약은 다음과 같이 이동할 수 있습니다.

1. 객실에 배정된 예약 카드를 **마우스로 드래그하여 분리**
2. 원하는 다른 객실로 이동하여 배정

분리된 예약 카드는 **다가오는 예약 카드와 동일하게 동작**합니다.

## 예약 재배정 (체크인 후 = 룸 체인지)

체크인 후 룸 체인지 시 기존 객실은 사용된 객실로 판단되어 이용만료로 변경됩니다.\
이용만료로 바뀐 객실은 객실 점검 / 정비 후 공실로 변경이 필요합니다.
