좌측 상단에 대표 키오스크를 클릭하여 제어 화면에 진입할 수 있습니다.\
2대 이상의 키오스크를 사용하는 업장은 대표 키오스크를 제외간 키오스크 제어 화면은 대시보드에서 노출되는 객실 하단에서 선택 가능합니다.

## 키오스크 제어 화면 진입

대시보드에서 제어하려는 **키오스크** 이름을 클릭하면 **키오스크 제어 화면**을 열 수 있습니다.\
해당 화면에서는 현재 운영 중인 키오스크의 상태를 확인하고 주요 기능을 원격으로 제어할 수 있습니다.

<Callout>
  <Tabs>
    <Tab title="장치관리">
      ![Image](/images/대시보드_키오스크_장치관리.png)

      키오스크 장치의 기본 상태 및 운영 설정을 관리합니다.

      - **소프트웨어 버전 확인**
      - **키오스크 재시작**
        - 키오스크 프로그램만 종료 후 재실행됩니다.
      - **판매 상태 설정**
        - 전체판매
        - 숙박만 판매
        - 대실만 판매
        - 판매중지
    </Tab>
    <Tab title="객실키방출">
      ![Image](/images/대시보드_키오스크_객실키방출.png)

      - 특정 객실의 **키를 원격으로 방출**할 수 있습니다.
      - 키 재발급 또는 긴급 상황 시 사용할 수 있습니다.
    </Tab>
    <Tab title="관리자 정책">
      ![Image](/images/대시보드_키오스크_관리자정책.png)

      키오스크에서 아래 기능의 **진입 가능 여부**를 설정합니다.

      - 관리자 모드
      - 키 발급 모드
      - 고급 관리 모드
      - 키 동기화 모드

      설정에 따라 현장에서 키오스크의 관리자 기능 접근을 제한하거나 허용할 수 있습니다.
    </Tab>
  </Tabs>
</Callout>
