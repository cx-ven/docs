# 예약 추가

대시보드 우측 상단의 **예약 추가** 버튼을 클릭하여 예약 추가 화면을 실행할 수 있습니다.\
예약 추가 화면에서 필요한 정보를 입력하여 예약을 직접 생성할 수 있습니다.

---

# 수기 예약 생성 기본 설정

### 1. 대실 포함 / 미포함

- 대실 판매를 하지 않는 경우 해당 옵션을 비활성화하여 **대실 관련 메뉴를 숨길 수 있습니다.**
- 설정은 **현재 사용 중인 장치에 저장되며**, 이후에도 동일하게 적용됩니다.

### 2. 객실 타입

- 예약할 **객실 타입**을 선택합니다.
- 패키지가 설정된 지점의 경우 **[패키지판매]** 체크 시 패키지 타입으로 예약할 수 있습니다.

### 3. 입실 / 퇴실 시간

- 예약의 입실 및 퇴실 시간을 설정합니다.

### 4. 기간 빠른선택, 이용시간 조절

- 기간 빠른선택/이용시간 조절 버튼으로 입/퇴실 시간을 편리하게 조절합니다.

### 5. 투숙 형태, 입실인원

- 예약의 투숙 형태 및 입실할 인원 수를 입력합니다.

### 6. 결제 정보

- 결제 금액
- 결제 구분 (선결제 / 현장결제 등)
- 미수금 입력 가능 ※ **[마일리지 제외]** 체크 시 해당 예약은 마일리지 적립 대상에서 제외됩니다.

### 7. 예약처 (플랫폼)

- 예약이 유입된 경로를 선택합니다.
- 플랫폼 목록 설정은 [컴퍼니 프로필 관리](https://ven-5828dcc8.mintlify.app/ko/profile/company)를 참고하세요.

### 8. 예약번호

- 키오스크 입실 시 사용할 **4자리 예약번호**를 입력합니다.
- OTA 예약의 경우 **예약번호 끝 4자리 입력을 권장**합니다.

### 9. 고객 정보

- 예약자명
- 연락처
- 차량 방문 여부
- 메모

### 10. 예약 옵션

- **예약문자 발송** 체크 시, 입력된 연락처로 예약 안내 문자가 발송됩니다.

  <Warning>
    1.예약옵션에 체크가 없을 경우 연락처가 있더라도 자동 문자가 발송되지 않습니다.

    2.자동 문자를 사용하지 않는 경우 문자가 발송되지 않습니다.

    3.자동 문자는 유료서비스 입니다.

    👉 [자동문자 바로가기](https://ven-5828dcc8.mintlify.app/ko/setting/SMS) 클릭
  </Warning>

모든 정보를 입력한 후 **[예약 추가하기]** 버튼을 클릭하면\
예약과 결제 내역이 함께 생성됩니다.
