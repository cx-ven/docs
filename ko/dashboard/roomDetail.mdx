# 객실 상세보기 <Badge color="green">사용 가능</Badge>

객실 상세보기 화면에서는 해당 객실의 상태, 예약 정보, 장치 제어 등 다양한 기능을 확인하고 관리할 수 있습니다.

---

## 1. 객실명 확인 / 객실키 발행

- 화면 상단에서 객실명을 확인할 수 있습니다.
- **객실키 발행** 버튼을 통해 키오스크에서 해당 객실의 키를 **수동으로 방출**할 수 있습니다
- **[영수증 포함]** 체크 시, 현재 입실 중인 예약의 영수증이 함께 출력됩니다.
- 영수증만 별도 출력/재출력은 어려우며, 객실 키도 한 번에 1개만 방출 됩니다.

---

## 2. 객실 상세보기 좌측 메뉴

객실 상세보기의 좌측 메뉴에서는 객실 장치 상태 확인, 음성 안내, 냉난방 제어, 객실 상태 변경, 체크인 처리, 배정 취소, 예약 정보 조회 등의 기능을 사용할 수 있습니다.

### 객실 장치 상태 확인, 음성 안내

<Accordion title="※ CCU를 연동하는 업장만 객실 장치 상태와 음성 안내가 활성화 됩니다.">
  <Callout>
    ※ [**현재상태] / [음성안내] / [냉난방제어]** 기능은 사용 중인 객실 관리 장치 종류(CCU)에 따라 지원되지 않을 수 있으며, 미지원 기능은 화면에 표시되지 않습니다.

    **객실 장치의 동작 상태를 아이콘으로 확인할 수 있습니다.**

    ### [현재상태]

    - 1~6번 램프
    - 객실키 전원
    - 문 열림/닫힘 상태

    현재 작동 중인 장치는 **주황색으로 표시**됩니다.

    ### [음성안내]

    - 음성 템플릿을 선택하여 해당 객실에 음성을 송출할 수 있습니다.
    - 사용 가능한 템플릿은 객실 관리 장치별로 제한됩니다.
    - 자주 사용하는 음성은 즐겨찾기로 등록하여 사용할 수 있습니다.

    ### [냉난방 제어]

    - 객실 에어컨 전원 ON/OFF
    - 난방 온도 조절
    - 객실 상태 변경
    - 체크인 표시
    - 배정 취소
    - 현재 예약 요약 정보 조회
  </Callout>
</Accordion>

### 객실 상태 변경

객실 상태 선택 박스를 통해 상태를 변경할 수 있습니다.

- **공실**: 현재 예약이 있을 경우 강제 퇴실 처리
- **전원 차단**: 객실 전원 차단 (일부 장치 미지원)
- **청소지시**: 청소 요청 (완료 시 기존 상태로 복구)
- **긴급청소**: 대시보드에서 강조 표시되는 청소 요청
- **숙박 / 대실**: 예약 유형 변경
- **청소중**: 키홀더 미설치 업장에서 수동 표시
- **판매중지**: 키오스크 판매 화면에서 제외
- **퇴실**: 사용 종료 처리 (예약/결제 취소되지 않음)
- **사용취소**: 예약 및 결제 전체 취소 후 상태 초기화

### 체크인 처리 / 배정 취소

- **체크인으로 표시하기**: 입실 대기 예약을 즉시 입실 처리
- **배정 취소**: 객실 배정을 해제하여 재배정 가능 상태로 변경

체크인 처리 시 키오스크 체크인 또는 키 삽입과 동일한 상태로 변경됩니다.

## 예약 요약 정보

좌측 하단에서 현재 객실에 배정된 예약의 요약 정보를 확인할 수 있습니다.

- 예약 정보 상세보기
- 회원 정보 상세보기
- 결제 정보 상세보기

예약 수정하기

1. 연필 아이콘이 표시된 항목 클릭 또는 [상세보기] 버튼 클릭
2. 내용 수정
3. **저장하기** 버튼 클릭

# 3. 빠른 기능 메뉴

빠른 기능 메뉴에서 다음 작업을 빠르게 수행할 수 있습니다.

<Tabs>
  <Tab title="객실 메모">
    - **객실 메모** : 해당 객실에 대해 남기는 메모 내용입니다. 현재 예약이 변경되거나 예약이 없을 때에도 항시 보존됩니다. \
      **예약 메모** : 현재 객실에 배정된 예약에 대해 남기는 메모입니다.현재 예약이 있는 경우에만 대시보드에서 표시되며, 예약 사용이 종료된 후에는 대시보드에서 사라지며 예약 탭의 상세보기 화면에서만 확인 가능합니다. \
      예약 메모와 객실 메모가 동시에 존재할 경우, 대시보드 상에서는 항상 예약 메모가 우선 표시됩니다.
  </Tab>
  <Tab title="이용중표시">
    - 해당 객실에 즉시 배정되는 예약을 바로 생성할 수 있습니다.
    - 예약 추가 방법과 동일합니다.
  </Tab>
  <Tab title="추가결제/시간변경">
    **추가결제/시간변경** 메뉴에서는 현재 객실에 배정된 예약의 퇴실 시간을 변경하고, 추가 결제 내역을 생성할 수 있습니다.

    1. **변경후 퇴실예정시간**\
       변경될 퇴실 시간을 표시합니다. 달력 아이콘을 눌러 직접 퇴실 시간을 선택할 수도 있습니다.
    2. **내역추가**\
       대실/숙박 추가 체크박스를 선택한 뒤 저장하면, 알맞는 새로운 예약 내역을 생성하여 통계 및 정산 내역상의 대실/숙박 개수를 1개 추가합니다.
    3. **연장/단축할 시간**\
       변경후 퇴실예정시간이 현재 퇴실 시간 기준으로 몇시간 단축/연장되는지 보기 쉽게 표시하고, 각각의 빠른 수정 버튼을 누르면 표시된 시간만큼 퇴실 시간을 간편하게 조절할 수 있습니다.
    4. **금액**\
       손님께서 결제하신 금액 혹은 결제해야 할 미수금 금액을 입력합니다. 해당 입력이 입력될 경우 현재 예약에 존재하는 결제 내역 하위에 알맞은 부속 결제 내역을 생성하여 결제 처리합니다.
    5. **결제구분**\
       손님께서 결제하신 결제 수단입니다. 결제 내역이 있을 경우에 선택합니다.
    6. **결제목적**\
       추가하고자 하는 결제 내역의 결제 목적을 자유롭게 입력하실 수 있습니다.

    필요한 항목을 입력한 뒤 [**저장**]버튼을 누르면 입력한 내용에 따라 퇴실 시간 변경 혹은 결제 내역 추가를 실행할 수 있습니다.
  </Tab>
</Tabs>

---

# 4. 예약/결제 내역 보기

현재 객실에 배정된 예약 및 해당 예약에 포함된 결제 내역을 표시하는 화면입니다. [상세보기] 버튼을 눌러 해당하는 예약 혹은 결제 내역 상세보기 화면으로 이동할 수 있습니다.
