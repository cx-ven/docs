# 예약 추가

예약 탭 → 우측 상단 **[예약추가]**

#### 기본 설정

- 대실 포함/미포함
- 객실 타입 또는 패키지 선택
- 입실 / 퇴실 시간

#### 편의 기능

- 기간 빠른 선택
- 이용시간 조절

#### 고객 및 이용 정보

- 투숙 형태 / 인원
- 예약자명 / 연락처 / 차량 여부 / 메모

#### 결제 정보

- 결제금액 / 결제수단
- 미수금 입력 가능
- 마일리지 제외 옵션

#### 기타

- 예약처(플랫폼)
- 예약번호(키오스크용 4자리)
- 예약문자 발송

입력 완료 후 **[예약 추가하기]** 클릭 시 예약과 결제 내역이 생성됩니다.

## 예약 수정 및 취소

- 예약 상세보기 → **[예약수정]**
- 수정 후 저장하면 즉시 반영됩니다.
- **[예약취소]** 클릭 시 예약이 취소됩니다.

※ 입력 항목은 예약 추가와 동일합니다.

# 단체 예약

예약 탭 → 우측 상단 **[단체예약]**

### 단체예약 생성 절차

1. 단체 기본정보 입력
2. 예약 그룹 추가
3. 예약 일괄 생성
4. **[단체예약생성]** 클릭

### 단체예약 생성 기본정보

- 단체명
- 예약처
- 담당자
- 입/퇴실 날짜
- 예약 상태 (가예약 / 확정 / 결제완료)

### 단체예약 생성

- 객실 타입별 수량 설정
- 기본 일정 범위 내에서만 설정 가능
- 생성 대기 목록에서 개별 수정 가능

### 단체예약 대표예약 일괄결제

- 전체 금액을 첫 예약에 합산
- 설정 후 취소 불가

### 단체예약 확인

- 예약 리스트에서 단체명 표시
- 개별 조회 및 수정 가능
- 단체 정보는 **일괄 수정 불가 (삭제만 가능)**
