# 단체 예약

예약 탭 → 우측 상단 **\[단체예약\]**

<img
  src="/images/예약_단체예약추가.png"
  alt="Image"
  title="Image"
  style={{ width:"60%" }}
/>

### 단체예약 생성 절차

1. 단체 기본정보 입력
2. 예약 그룹 추가
3. 예약 일괄 생성
4. **\[단체예약생성\]** 클릭

### 단체예약 생성 기본정보

- 단체명
- 예약처
- 담당자
- 입/퇴실 날짜
- 예약 상태 (가예약 / 확정 / 결제완료)

### 단체예약 생성

- 객실 타입별 수량 설정
- 기본 일정 범위 내에서만 설정 가능
- 생성 대기 목록에서 개별 수정 가능

### 단체예약 대표예약 일괄결제

- 전체 금액을 첫 예약에 합산
- 설정 후 취소 불가

### 단체예약 확인

- 예약 리스트에서 단체명 표시
- 개별 조회 및 수정 가능
- 단체 정보는 **일괄 수정 불가 (삭제만 가능)**
