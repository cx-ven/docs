# 예약 추가

예약 탭 → 우측 상단 **[예약추가]**

👉 [<kbd>예약 추가 기능 매뉴얼 바로가기</kbd>](https://ven-5828dcc8.mintlify.app/ko/dashboard/res)

## 예약 수정 및 취소

- 예약 상세보기 → **[예약수정]**
- 수정 후 저장하면 즉시 반영됩니다.
- **[예약취소]** 클릭 시 예약이 취소됩니다.

※ 입력 항목은 예약 추가와 동일합니다.

# 단체 예약

예약 탭 → 우측 상단 **[단체예약]**

### 단체예약 생성 절차

1. 단체 기본정보 입력
2. 예약 그룹 추가
3. 예약 일괄 생성
4. **[단체예약생성]** 클릭

### 단체예약 생성 기본정보

- 단체명
- 예약처
- 담당자
- 입/퇴실 날짜
- 예약 상태 (가예약 / 확정 / 결제완료)

### 단체예약 생성

- 객실 타입별 수량 설정
- 기본 일정 범위 내에서만 설정 가능
- 생성 대기 목록에서 개별 수정 가능

### 단체예약 대표예약 일괄결제

- 전체 금액을 첫 예약에 합산
- 설정 후 취소 불가

### 단체예약 확인

- 예약 리스트에서 단체명 표시
- 개별 조회 및 수정 가능
- 단체 정보는 **일괄 수정 불가 (삭제만 가능)**
