# 객실 상태 팔레트

- 우상단 팔레트 모양의 아이콘을 누르면 왼쪽부터 [청소지시, 청소중, 공실, 판매중지]에 해당하는 버튼이 표시됩니다.
- 원하는 버튼을 눌러 마우스 모양이 붓 표시로 변한 뒤, 상태 변경을 원하는 객실 카드를 클릭 시 객실의 상태가 변경됩니다.
- 편리하게 접근할 수 있는 단축키 기능을 제공하고 있습니다.

  | 단축키 | 기능             |
  | --- | :------------- |
  | Q   | 청소지시           |
  | W   | 청소중            |
  | E   | 공실             |
  | R   | 판매중지           |
  | S   | 대시보드 검색 입력창 선택 |
  | A   | 예약 추가하기        |

# 난방, 에어컨 제어

클라우드에서 객실의 냉난방을 원격으로 제어할 수 있습니다.\
설정한 온도는 선택한 객실에 **즉시 적용됩니다.**

- 냉난방 **ON/OFF 및 온도 변경** 기능 제공
- 제어 가능한 객실에서만 UI가 표시됩니다.
- 해당 기능은 예약 설정이 아닌 **즉시 적용되는 수동 제어**입니다.

<Warning>
  사용 전 확인하세요

  - **CCU 사용 업장에서만 제공되는 기능**입니다.
  - 객실의 에어컨 또는 난방기가 **키텍(RCU)에 연결된 경우에만 제어할 수 있습니다.**
  - **온돌(바닥 난방)은 지원하지 않습니다.**

  <Accordion title="지원 키텍 브랜드">
    - 시리얼
    - 헤르메스
    - 더엠알

    ※ 위 브랜드 외 키텍은 클라우드를 통한 객실 온도 제어를 지원하지 않습니다.
  </Accordion>
</Warning>

<Tip>
  객실의 문 열림/닫힘 등 상태 변경이 발생하면 설정한 온도가 유지되지 않을 수 있습니다.\
  기본 온도 설정이 있는 경우 이후 해당 온도로 다시 적용될 수 있습니다.
</Tip>

# 대시보드 설정

대시보드 우측 상단 설정(톱니바퀴) 메뉴를 클릭 시 대시보드 화면에서 보여지는 내용에 대해 설정을 변경할 수 있습니다.

 1. **방 상태영역 확대비율** 대시보드에 노출되는 객실 상태영역의 크기를 작거나 크게 비율 설정하여 확인하실 수 있습니다.
 2. **객실 표시 방식** 대시보드에 노출되는 객실의 표시 방식을 설정할 수 있습니다. 전체/층별/객실타입 별 중에서 설정이 가능합니다.
 3. **객실 정렬순서** 대시보드에 노출되는 객실의 정렬 방식을 설정할 수 있습니다. 내림차순(객실호수가 낮은 순)/오름차순(객실호수가 높은 순) 중에서 설정이 가능합니다.
 4. **예약 표시 타입** 다가오는 손님 예약 건에 대해 대시보드 내 표시 형태를 세로/가로 중에서 설정 가능합니다. 익일 예약 숨기기 :  익일 예약 숨기기 활성화 시 다가오는 예약 리스트에 익일 예약이 보이지 않도록 설정 가능합니다. 대실/숙박 예약 구분표시 : 활성화 시 대실과 숙박이 두 줄로 다가오는 예약에 보여집니다.
 5. **객실별 시간 표시** 대시보드 내 객실 별 시간 표시를 남은 시간 또는 사용시간 으로 표시하도록 설정 가능합니다.
 6. **객실별 메모표시** 대시보드에 표기될 메모의 우선순위를 설정 가능합니다.
 7. **객실 정렬순서** 대시보드에서 보여지는 객실의 정렬 순서를 설정할 수 있습니다.
 8. **예약 표시타입** 다가오는 예약를 하단 가로식 또는 우측 세로식으로 설정할 수 있습니다.
 9. **객실별 시간표시** 각 객실 별 숙박중인 시간을 남은 시간 또는 사용 시간이 노출될 수 있도록 설정할 수 있습니다.
10. **객실별 메모표시** 저장한 메모가 대시보드에 노출되는 우선순위를 설정할 수 있습니다.
11. **객실필터** 판매중지된 객실 표시 : 판매중지 상태의 객실도 대시보드에 표시하도록 설정 가능합니다. 예비객실 표시 : 고객용이 아닌 벤디트 용(객실명에 “예비\_”를 붙여 대시보드에서 표시를 제한할 수 있도록 설계한 기능) 동별로 그룹화하여 표시 : 건물에 따라 동이 나눠져있는 경우 해당 동별로 그룹화하여 객실을 보여주도록 설정이 가능합니다. 　　　　　　　　　　  설정-\>객실 : 객실명(호수)을 A동-101 호 등으로 설정하신 뒤 이용하실 수 있습니다. 체크아웃 30분 전 알림 미표시 검색 시 모든 객실 표시
12. **객실정보 :  CCU 사용 업장에 한 해 설정 가능합니다.** 대시보드 내 통신상태(깜빡임)과 객실마다 온도 표시 노출 여부를 설정할 수 있습니다. 사용하는 CCU별로 통신상태가 깜빡거리거나 깜빡거리지 않을 수 있습니다. → 표 참고(https://www.notion.so/vendit/CX-f2cbda41eb514390a47a73e073a22cda?source=copy_link#26fa1c39fbb8804ba9b5d24771b0f949)
13. **대체 테마** 객실 시인성 모드 색각이상용 색구분 사용 : 해당 메뉴는 설정 후 새로고침(F5) 하셔야 반영됩니다.
