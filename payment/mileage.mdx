## 마일리지 기능 설정

마일리지 내역 메뉴에서는 고객의 마일리지 적립 및 사용 내역을 확인할 수 있습니다.

마일리지 적립 방법

- 키오스크 현장 결제 후 휴대폰 번호 입력 시 자동 적립
- 관리자 수동 적립

※ 마일리지 기능을 사용하려면 아래 경로에서 기능을 활성화해야 합니다.\
**경로:** 클라우드 \> 설정 \> 판매 및 마일리지 설정

## 마일리지 수동 적립

1. 화면 오른쪽 상단 **[+수동적립]** 클릭
2. 휴대폰 번호와 적립할 마일리지 입력
3. [적립] 클릭

추가 기능

- **추가입력** 버튼으로 여러 고객 동시 적립 가능
- 키보드 **Tab** 키로 입력 항목 간 이동 가능

## 마일리지 상태 확인 및 취소

- 적립 리스트에서 **[적립취소]** 클릭 시 마일리지 취소
- 상태 표시
  - **취소됨**: 적립 취소된 경우
  - **사용됨**: 고객이 사용한 경우
