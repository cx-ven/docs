title: "쿠폰 내역  "
---

쿠폰 기능은 **부킹엔진이 활성화된 지점에서만 사용 가능**합니다.\
고객이 부킹엔진 예약 시 할인 혜택을 받을 수 있도록 쿠폰을 생성하고 발행할 수 있습니다.

## 쿠폰 발행

1. **[새 쿠폰 발행]** 클릭
2. 쿠폰 정보 입력
3. **[발행]** 클릭

설정 항목

**발행 매수**

- 발행할 쿠폰 수량 입력

**쿠폰 타입**

- 지점 공통 쿠폰
- 객실 타입 한정 쿠폰

**할인 타입**

- 퍼센트 할인
- 금액 할인

**최대 할인금액**

- 퍼센트 할인 시 최대 할인 금액 설정

**최소 사용금액**

- 설정 금액 미만 결제 시 사용 불가

**쿠폰 만료기간**

- 만료일 이후 사용 불가

**1매당 최대 사용 횟수**

- 하나의 쿠폰 코드 재사용 가능 횟수
- 제휴/프로모션용 공통 코드 발급 시 활용

발행 완료 후 쿠폰 코드는 **엑셀 파일로 다운로드**됩니다.

---

## 쿠폰 사용

고객은 부킹엔진 예약 결제 화면에서 쿠폰 코드를 직접 입력하여 할인 적용할 수 있습니다.
