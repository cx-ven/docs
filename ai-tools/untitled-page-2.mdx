## 연동 옵션

주요 기능

- 예약 자동 객실 배정
- RCU 미연동 시 체크인 상태 자동 설정
- 이상 사용기록 저장
- 체크인 시간 제한 해제
- 수동 배정 시 입퇴실 시간 자동 조정
- 전체 객실 자동 체크아웃(게스트하우스)
