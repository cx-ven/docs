---
title: "계정 및 일반"
---

1. 지점 문의전화 번호 :  키오스크 화면 아래쪽에 노출되는 문의전화 번호를 설정 하실 수 있습니다. \*변경시 키오스크 관리자모드 진입 시에 필요한 비밀번호도 변경이 됩니다.
2. 기본 체크인/체크아웃 시간을 설정할 수 있습니다. 설정 → 객실에서 설정된 체크인/체크아웃 시간이 없다면 해당 시간을 따라갑니다.
3. 알림메일 수신주소 : 성인인증 시, 손님이 제출한 신분증/사진이 첨부되어 메일로 전송됩니다.
4. 알림문자 수신번호 : 키오스크 현장 결제 시, 성인인증 직원승인 시, 자동문자 충전 알림, 공지사항 등을 수신할 수 있습니다.
5. 핀번호 로그인 : V-CLOUD 앱(안드로이드만 지원) 로그인 시 사용  합니다,
6. 비밀번호 변경 : 현재 비밀번호를 입력 한 후 변경된 비밀번호 정책에 따라 비밀번호 변경이 가능합니다.
7. 전체 기기 로그아웃 : 로그인 되어있는 기기들에서 전체 로그아웃을 진행합니다.
