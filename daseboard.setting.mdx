| 상세보기  좌측 메뉴에서는 객실 장치 현재 상태, 음성 안내, 냉난방 제어, 객실 상태 변경, 체크인 표시, 배정취소, 현재 예약 요약 정보 조회 등의 기능을 이용하실 수 있습니다.  ![][image7]           \<그림. 좌측메뉴 상단 장치 관련 메뉴 \> **(장치) 현재상태, 음성안내, 냉난방제어** 메뉴의 경우 지점에서 사용중인 객실 관리 장치의 종류에 따라 각각 기능을 지원하지 않는 경우가 존재합니다. 이 경우 지원되지 않는 메뉴는 상세보기 화면에서 노출되지 않습니다. **현재상태**현재상태 메뉴에서는 우측에 표시되는 8개의 아이콘을 통해 현재 객실에 설치된 장치가 동작중인지 아닌지를 확인할 수 있습니다.각각의 아이콘은 왼쪽에서부터 **1~6번 램프** (**램프 아이콘**), **객실키 전원** (**전기 아이콘**), **문 열림/닫힘** (**문 아이콘**)으로 표시됩니다. 현재 작동중인 장치는 **주황색**으로 강조 표시됩니다.  **음성안내** 음성안내 메뉴에서는 재생 가능한 음성 템플릿을 선택하여 해당 객실에 음성 송출을 명령할 수 있습니다. 재생 가능한 음성 템플릿은 객실 관리 장치별로 정해져 있는 목록 안에서만 선택 가능합니다.또, 자주 쓰는 음성의 경우 즐겨찾기에 추가하여 사용하실 수 있습니다. 즐겨찾기 설정의 경우 **4.4 키오스크 설정** 도움말을 참고하세요. **냉난방제어**냉난방제어 메뉴에서는 각 객실별로 에어컨 전원 켜기/끄기, 현재 설정된 난방 온도 조절 기능을 사용하실 수 있습니다.  ![][image8]           \<그림. 좌측 메뉴 중단 현재 상태 표시 셀렉트박스\> 객실의 현재 상태를 표시하는 셀렉트 박스를 눌러, 현재 상태를 원하는 상태로 변경할 수 있습니다. **공실**객실 상태를 공실로 전환합니다. 현재 예약이 존재할 경우 강제 퇴실 처리합니다. **전원 차단**객실의 전원을 차단합니다. 객실 관리 장치에 따라 미지원 장치에서는 노출되지 않습니다. **청소지시**객실 청소를 지시합니다. 현재 사용중인 객실의 상태 위에 덮어 씌울 수 있습니다. 이 경우 청소 완료시 원래의 상태로 복구됩니다. **긴급청소**청소지시와 같이 동작하나, 긴급한 청소 건임을 알리기 위해 대시보드에서 강조 표시됩니다. **숙박/대실**객실에 현재 예약이 배정되어 있을 경우 예약의 숙박/대실 타입을 선택한 타입으로 변경합니다. 객실이 현재 공실일 경우 이용중표시 메뉴를 표시합니다. **청소중**키홀더 미 설치 업장의 경우 해당 객실을 현재 청소중임을 수동으로 표시합니다. **판매중지**해당 객실의 판매를 중지하여 키오스크 판매 화면에서 노출되지 않도록 설정합니다. 현재 사용중인 객실도 판매 중지할 수 있습니다. **퇴실**현재 객실이 사용중일 경우 퇴실 처리합니다. 예약 혹은 결제내역은 취소되지 않습니다. **사용취소**현재 객실에 입실한 예약 및 결제 내역을 전부 취소하고, 객실이 입실 대기일 경우 공실로, 객실이 이미 사용중일 경우 퇴실 상태로 상태를 전환합니다.  ![][image9]           \<그림. 좌측 메뉴 중단 체크인으로 표시하기 / 배정 취소 버튼\> 체크인으로 표시하기, 배정 취소 버튼을 통해 현재 **입실 대기중인** **예약**을 **입실 처리**하거나, **객실 배정을 취소**할 수 있습니다. 입실 처리된 예약은 키오스크를 통해 체크인하거나 객실 키를 삽입한 것과 같이 입실 상태로 전환 표시되며, 배정 취소된 예약은 재 배정이 가능한 상태로 되돌려집니다. ![][image10]           \<그림. 좌측 메뉴 하단 예약 요약 정보 보기 메뉴\> 좌측 메뉴 하단에서는 현재 객실에 배정된 예약의 요약 정보를 확인하거나 수정할 수 있습니다. **예약정보 조회**, **회원정보 조회** 버튼을 통해 현재 예약 정보, 등록된 회원 정보 상세보기 화면으로 이동할 수 있습니다. 우측에 연필 아이콘이 표시되어 있는 정보를 좌클릭하면 정보를 수정 입력할 수 있으며, 정보 수정 이후 **저장하기** 버튼을 눌러 수정된 정보를 저장합니다.  ![][image11]           \<그림. 요약정보 수정 / 저장\> |

| ![][image12]           \<그림. 빠른기능 메뉴\> 빠른 기능 메뉴에서는 **객실메모, 이용중표시, 추가결제/시간변경** 메뉴를 각각 선택하여 사용할 수 있습니다. ![][image13]          \<그림. 객실메모 메뉴\> **객실 메모** 메뉴에서는 현재 선택한 객실 및 현재 객실에 배정된 예약에 대한 **메모를 추가, 수정, 삭제**할 수 있습니다. **객실 메모**는 해당 객실에 대해 남기는 메모 내용입니다. 현재 예약이 변경되거나 예약이 없을 때에도 항시 보존됩니다.**예약 메모**는 현재 객실에 배정된 예약에 대해 남기는 메모입니다. 현재 예약이 사용중일 때에만 대시보드에서 표시되며, 예약 사용이 종료된 후에는 예약 탭의 상세보기 화면에서만 확인 가능합니다. **예약 메모와 객실 메모가 동시에 존재할 경우, 대시보드 상에서는 항상 예약 메모가 우선 표시됩니다.**  \<그림. 대시보드 객실 정보의 메모 표시 화면\>  ![][image15]          \<그림. 이용중표시 메뉴\> **이용중표시** 메뉴에서는 해당 객실에 즉시 배정되는 예약을 바로 생성할 수 있습니다. 해당 화면에서 예약을 생성하는 방법은 일반적인 예약 추가와 같습니다. 이에 대한 설명은 **1.3 예약 추가하기** 도움말을 참고하세요.![][image16]           \<그림. 추가결제/시간변경 메뉴\> **추가결제/시간변경** 메뉴에서는 현재 객실에 배정된 예약의 퇴실 시간을 변경하고, 추가 결제 내역을 생성할 수 있습니다. **변경후 퇴실예정시간**변경될 퇴실 시간을 표시합니다. 달력 아이콘을 눌러 직접 퇴실 시간을 선택할 수도 있습니다. **내역추가**대실/숙박 추가 체크박스를 선택한 뒤 저장하면, 알맞는 새로운 예약 내역을 생성하여 통계 및 정산 내역상의 대실/숙박 개수를 1개 추가합니다. **연장/단축할 시간**변경후 퇴실예정시간이 현재 퇴실 시간 기준으로 몇시간 단축/연장되는지 보기 쉽게 표시하고, 각각의 빠른 수정 버튼을 누르면 표시된 시간만큼 퇴실 시간을 간편하게 조절할 수 있습니다. **금액**손님께서 결제하신 금액 혹은 결제해야 할 미수금 금액을 입력합니다. 해당 입력이 입력될 경우 현재 예약에 존재하는 결제 내역 하위에 알맞은 부속 결제 내역을 생성하여 결제 처리합니다. **결제구분**손님께서 결제하신 결제 수단입니다. 결제 내역이 있을 경우에 선택합니다. **결제목적**추가하고자 하는 결제 내역의 결제 목적을 자유롭게 입력하실 수 있습니다. 필요한 항목을 입력한 뒤 [**저장**]버튼을 누르면 입력한 내용에 따라 퇴실 시간 변경 혹은 결제 내역 추가를 실행할 수 있습니다. |
