# 객실 상태

객실들 및 해당 객실들에 배정된 예약의 상태가 **객실 카드**로 표시됩니다.

<Tabs>
  <Tab title="공실">
    현재 예약이 배정되어 있지 않고, 즉시 사용 가능한 객실입니다.  공실 객실 중 요금 설정을 통해 현재 시간에 판매 요금이 책정되어 있는 공실 객실이 키오스크 판매 화면에 노출됩니다.
  </Tab>
  <Tab title="청소">
    청소를 지시한 객실은 청소지시 상태로 표시됩니다. 실제 청소부가 입실하여 청소를 진행 중인 객실은 청소중 상태로 강조 표시됩니다.
  </Tab>
  <Tab title="숙박 ">
    숙박 예약이 입실하여 이용중이거나 입실 대기 상태인 객실은 숙박중 혹은 입실대기 상태로 표시됩니다.
  </Tab>
  <Tab title="대실">
    대실 예약이 입실하여 이용중이거나 입실 대기 상태인 객실은 대실중 혹은 입실대기 상태로 표시됩니다
  </Tab>
  <Tab title="이용만료">
    예약 이용 시간이 지난 객실은 [이용만료] 상태로 표시됩니다. \
    설정 -\> 예약 설정에서 설정한 객실 카드키 제거시 퇴실로 판단 설정에 따라이용 만료 시각이 임박했고, 객실 카드키가 제거된 상태의 객실의 경우 [퇴실확인] 상태로 표시됩니다.
  </Tab>
  <Tab title="판매중지">
    객실을 선택 후 판매중지 상태를 선택했거나, 설정 -\> 판매 및 마일리지 설정에서 판매 중지 상태를 선택한 객실들은 회색 판매중지 상태로 표시됩니다.  \
    현재 입실 중인 객실, 청소중인 객실도 판매 중지할 수 있습니다.
  </Tab>
  <Tab title="숙박(장기)">
    수기 예약 생성 시 장기 숙박으로 선택할 수 있습니다.\
    숙박과 장기 숙박을 판별하기 위해 노출되는 색상이 다릅니다.
  </Tab>
</Tabs>

# 다가오는 예약

대시보드의 다가오는 예약 영역에서 오늘 입실 예정인 예약을 확인하고, 드래그하여 원하는 객실에 배정할 수 있습니다.

> - 익일 예약은 당일 시각의 24시간 내의 익일 입실 예정인 예약이 노출됩니다.
> - 다가오는 예약 카드 위에 🖱️마우스를 올린 뒤 마우스 휠을 움직이면 더 많은 예약을 볼 수 있습니다.

다가오는 예약이 여러 건일 경우, 입실 시간을 기준으로 **30분 단위 예약 그룹**이 표시됩니다.

> - 각 버튼에 해당 시간대의 예약 수가 함께 표시됩니다.
> - 원하는 시간대 버튼을 선택하면, **해당 시간대의 예약만 필터링**하여 확인할 수 있습니다.

# 객실 배정 방법

<Steps>
  <Step title="다가오는 예약으로 노출 된 예약 카드를 마우스로 클릭한 상태에서 드래그합니다.">
    
  </Step>
  <Step title="원하는 객실 카드 위에 놓으면 배정됩니다.">
    ### 객실 상태 표시 기준

    예약 이동 시 객실 상태에 따라 다음과 같이 표시됩니다.

    - **매우 어두움**: 배정 불가
    - **약간 어두움**: 타입 불일치 (배정 시 확인 창 표시)
    - **밝게 표시**: 배정 가능
      - 공실: 즉시 배정
      - 기존 예약 있음: 예약 병합 확인 후 처리
  </Step>
  <Step title="예약 재배정">
    이미 배정된 예약은 드래그하여 다른 객실로 이동할 수 있습니다.

    이동된 예약 카드는 다가오는 예약 카드와 동일하게 동작합니다.
  </Step>
</Steps>

<Tip>
  타입이 일치하지 않는 객실에 배정할 경우, 확인 창에서 최종 승인 후 배정됩니다.
</Tip>

# FAQ

<Accordion title="Q. 고객님은 키오스크에서 키를 반납하지 않았다고 하는데요?">
  1. 청소 직원이 수거해 둔 객실 키를 한 번에 체크아웃 처리했는지 확인합니다.
  2. 키오스크에서 키 반납은 하지 않았지만, 업장에서 클라우드에서 직접 퇴실 처리했는지 확인합니다.
  3. 다른 고객 또는 직원이 분실된 키를 주워 반납했는지 확인합니다.
</Accordion>

<Accordion title="Q. 아무도 입실하지 않았는데 객실 상태가 키가 꽂혀 있는 것으로 표시됩니다.">
  1. **키 홀더 내부에 이물질이나 먼지**로 인해 센서가 가려졌을 수 있으므로 키 홀더 내부를 청소합니다.
  2. **키텍 통신 이상**일 수 있으므로, 해당 객실 차단기를 내린 후 **10초 뒤 다시 올려** 재부팅합니다.
  3. 객실이 **상시 전원 공급 상태**인지 확인합니다.
  4. 키가 없어도 키텍 프로그램의 **지연(Delay) 설정**으로 전원 공급 신호가 전달되고 있을 수 있습니다.
  5. 키텍 프로그램 통신이 중단된 경우 **마지막 상태가 유지**될 수 있습니다. 사용 중인 **키텍 업체를 확인**하여 점검 및 조치를 진행합니다.
</Accordion>

<Accordion title="Q. 자동으로 청소지시로 변경되지 않고 ‘이용만료’로 표시됩니다.">
  - **자동 청소지시 기능이 비활성화**되어 있는지 확인합니다.
  - 퇴실 시간이 지났더라도 **키오스크에서 체크아웃 처리하지 않은 경우** 상태가 변경되지 않을 수 있습니다.
  - 설정 경로 : **설정 → 키오스크** 에서 **자동 청소지시 비활성화** 체크가 되어있는지 확인합니다.
</Accordion>
