| 지점 대시보드 화면에서 현재 동작 중인 키오스크 및 해당 키오스크에서 관리 중인 객실의 요약 현황을 확인할 수 있습니다. ![][image3] \<그림. 키오스크 상태, 객실 상태 요약 \> 키오스크 상태 표시 화면 아래에 해당 키오스크에서 관리 중인 객실들 및 해당 객실들에 배정된 예약의 상태가 **객실 카드**로 표시됩니다. ![][image4] \<그림. 대시보드 객실 상태 표시 화면 \> 각 객실의 상태는 아래와 같은 기준에 따라 표시합니다. 공실현재 예약이 배정되어 있지 않고, 즉시 사용 가능한 객실입니다. 키오스크를 통해 객실을 판매중일 경우, 공실 상태인 객실만이 키오스크에 표시됩니다. 다만, 모든 공실 객실이 키오스크에 표시되는 것은 아니며, 공실 객실 중 요금 설정을 통해 현재 시간에 판매 요금이 책정되어 있는 객실만이 키오스크 판매 화면에 노출됩니다. **판매중지**객실을 선택 후 판매중지 상태를 선택했거나, 설정 \> 판매 및 마일리지 설정에서 판매 중지 상태를 선택한 객실들은 회색 판매중지 상태로 표시됩니다.현재 입실 중인 객실, 청소중인 객실도 판매 중지할 수 있습니다. **청소지시 / 청소중**청소를 지시한 객실은 청소지시 상태로 표시됩니다. 실제 청소부가 입실하여 청소를 진행 중인 객실은 청소중 상태로 강조 표시됩니다. **숙박중 (입실대기)**숙박 예약이 입실하여 이용중이거나 입실 대기 상태인 객실은 숙박중 혹은 입실대기 상태로 표시됩니다. **대실중 (입실대기)**대실 예약이 입실하여 이용중이거나 입실 대기 상태인 객실은 대실중 혹은 입실대기 상태로 표시됩니다. **이용만료 / 퇴실확인**예약 이용 시간이 지난 객실은 [이용만료] 상태로 표시됩니다.설정 \> 예약 설정에서 설정한 객실 카드키 제거시 퇴실로 판단 설정에 따라 이용 만료 시각이 임박했고, 객실 카드키가 제거된 상태의 객실의 경우 [퇴실확인] 상태로 표시됩니다. 객실 상태 표시 화면에서 원하는 객실을 클릭해 해당 객실의 상세보기 화면을 표시할 수 있습니다. ![][image5] \<그림. 대시보드 객실 상세보기 화면 \> 객실 상세보기 화면에서는 아래와 같은 세부 화면들을 확인하실 수 있습니다. **객실명 확인 / 객실키 발행**객실키 발행 버튼을 통해 키오스크에서 해당 객실의 객실 키를 수동 방출하도록 명령하실 수 있습니다. [영수증 포함] 체크박스를 체크하여 객실 키를 방출할 경우 현재 객실에 입실 중인 예약의 영수증을 함께 인쇄하여 객실 키를 방출합니다. **좌측 메뉴**상세보기 좌측 메뉴에서는 객실 장치 상태 확인, 음성 안내, 냉난방 제어, 객실 상태 변경, 체크인 표시, 배정취소, 현재 예약 요약 정보 조회 등의 기능을 이용하실 수 있습니다.해당 메뉴의 상세한 사용 방법은 **1.2.1 객실 상세보기 좌측 메뉴** 도움말을 참고하세요. **빠른 기능 메뉴**빠른 기능 메뉴 화면에서는 객실 메모 설정, 객실 이용중 표시, 추가결제 / 시간 변경 메뉴를 사용 가능합니다.해당 메뉴의 상세한 사용 방법은 **1.2.2 객실 상세보기 빠른 기능 메뉴** 도움말을 참고하세요. **예약/결제 내역 보기**현재 객실에 배정된 예약 및 해당 예약에 포함된 결제 내역을 표시하는 화면입니다. [**상세보기**] 버튼을 눌러 해당하는 예약 혹은 결제 내역 상세보기 화면으로 이동할 수 있습니다. **객실 사용기록**사용기록 화면에 표시되는 사용기록 중, 해당 객실에 해당하는 사용기록만 간추려 볼 수 있습니다. 자주 변경하는 객실 상태인 [**청소지시, 청소중, 공실**] 상태의 경우 사용 편의를 위해 **객실** **상세보기** 화면까지 들어가지 않고, 대시보드 상에서 객실 상태를 편리하게 변경할 수 있는 **객실 상태 팔레트** 기능을 제공하고 있습니다.  **![][image6]**  \<그림. 객실 상태 팔레트\> **객실 상태 팔레트**는 대시보드 우 상단의 기능 박스 중 **팔레트** 모양의 아이콘을 눌러 사용할 수 있습니다. 팔레트를 누르면 **왼쪽부터 각각 [청소지시, 청소중, 공실]**에 해당하는 버튼이 표시됩니다. 원하는 상태에 해당하는 버튼을 눌러 커서 모양이 붓 표시로 변한 뒤 **상태 변경을 원하는 객실 카드를 누르면** 객실의 상태가 원하는 상태로 변경됩니다. 이 기능은 단축키를 통해 편하게 이용하실 수도 있습니다. 단축키에 대한 설명은 **1.5 단축키** 도움말을 참고하세요. |
